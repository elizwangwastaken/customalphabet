# about

this project is a custom handwriting-to-font web application that allows users to digitally draw their own alphabet and instantly use it as a functional typeface

you can export your creation as a professional .otf font file, making your personal handwriting available for use in any external design software or document editor :)

try it here: https://elizwangwastaken.github.io/customalphabet/
